# 👨‍💻 Personal Portfolio Website

## 📌 Project Overview

This project is a responsive personal portfolio website developed using HTML and CSS. It showcases my profile, technical skills, certifications, academic achievements, and contact information in a clean and professional layout.

This project was created as **Task 2** of the **Oasis Infobyte Web Development and Designing Internship**.

---

## 🎯 Objective

The objective of this project is to design a personal portfolio website that represents my skills, education, achievements, and contact details while demonstrating my understanding of HTML and CSS.

---

## ✨ Features

* Responsive Navigation Bar
* Hero Section
* About Me
* Skills Section
* Academic Statistics
* Certifications
* Contact Information
* Responsive Design
* Hover Effects
* Smooth Scrolling

## 🛠 Technologies Used
